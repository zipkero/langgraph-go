# textbook-parity SPEC

## 1. 범위
- hanbit-aiagent 교재(파이썬 원본)를 치환 규칙 없이 이 레포로 따라갈 수 있도록, 교재와 다른 세 지점을
  교재와 동일한 선택지로 맞춘다:
  - OpenAI 챗 모델 지원 (교재: gpt-4o 계열)
  - OpenAI 임베딩 지원 (교재: text-embedding-3-small, 1536차원)
  - Chroma 벡터스토어 지원 (교재: CHAP6·7 로컬 벡터스토어 실습)
- Supabase 벡터 검색 경로를 1536차원 기준으로 전환해 교재의 SQL 예제를 수정 없이 사용할 수 있게 한다.
- 레포 문서(README)의 기본 프로바이더 기준을 OpenAI로 갱신한다.

## 2. 목표
- 학습자가 교재의 어느 챕터를 펴도 "이 레포에서는 무엇으로 바꿔 쓴다"는 규칙을 기억할 필요 없이,
  교재가 지정하는 프로바이더·모델·벡터스토어 그대로 실습을 진행할 수 있다.
- 기존 Anthropic·Ollama·Supabase(768) 사용자는 명시적으로 대체 경로를 지정해 계속 사용할 수 있다
  (기본 기준만 OpenAI로 바뀌며 기존 구현은 제거하지 않는다).

## 3. 제약
- 기존 공개 인터페이스(챗 클라이언트·임베딩 클라이언트·벡터스토어 계약)는 하위호환을 유지한다.
  구현체 추가와 기본값·문서 변경만 허용하고, 기존 호출 코드를 깨는 시그니처 변경은 하지 않는다.
- OpenAI 경로는 API 키 환경변수가 있어야 동작하며, 키가 없는 환경에서도 나머지 기능의 빌드·테스트는
  깨지지 않아야 한다.
- Chroma 경로는 외부에서 실행 중인 Chroma 서버(예: 로컬 Docker)를 전제하며, 서버 미기동 환경에서도
  나머지 기능의 빌드·테스트는 깨지지 않아야 한다.
- 교재가 기준으로 삼는 버전 계열(langchain 1.0 세대 API, chromadb 1.x, text-embedding-3-small
  1536차원)과의 대응을 기준으로 한다.
- Supabase 차원 전환에 따라 기존 768차원 데이터는 폐기하고 재적재한다(2026-07-02 사용자 확정).

## 4. 제외 범위
- OpenAI 외 추가 프로바이더(Gemini 등) 지원.
- 768/1536 이원 스키마 병행 지원 — 1536 전환으로 단일화하며 768 전용 경로는 유지하지 않는다.
  (Ollama 임베딩 클라이언트 자체는 유지하되, Supabase 기본 스키마의 기준 차원은 1536이다.)
- 교재 챕터별 Go 예제 코드 작성(B안) — 별도 feature로 진행한다.
- interrupt/HITL, Postgres/SQLite 체크포인터 등 교재에 등장하지 않는 기능.
- langgraph-cli / LangGraph Studio 대응물(시각화 개발 서버).
- Chroma 서버 자체의 설치·구동 자동화(문서 안내는 범위 내, 도구화는 범위 외).

## 5. 완료 조건
1. provider 스펙 문자열 `openai:<model>`로 초기화한 챗 클라이언트가 일반 대화, 도구 바인딩·호출,
   토큰 스트리밍, 구조화 출력 요청 각각에 대해 기존 챗 클라이언트 계약과 동일한 형태의 응답을
   반환한다.
2. provider 스펙 문자열 `openai:text-embedding-3-small`로 초기화한 임베딩 클라이언트가 1536차원
   벡터를 반환한다.
3. Chroma 서버를 백엔드로 지정한 벡터스토어가 문서 추가, 유사도 검색, retriever 변환 호출에서
   기존 벡터스토어 계약과 동일한 형태의 결과를 반환한다.
4. OpenAI 임베딩(1536차원)으로 적재한 문서가 Supabase 벡터 검색 경로에서 정상 조회되며, 이때
   사용하는 DB 스키마·검색 함수가 교재의 SQL 예제와 차원·시그니처가 일치한다.
5. 기존 Anthropic 챗·Ollama 임베딩·인메모리 벡터스토어 경로의 기존 테스트가 전부 통과한다
   (회귀 없음).
6. README에 OpenAI가 기본 기준 프로바이더로 기재되고, 교재 대비 치환 없이 대응됨과 대체 경로
   (Anthropic/Ollama) 사용법이 문서화되어 있다.
